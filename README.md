> **Original Launch:** March 2026

### 🚀 **About the Project**

A modern, secure, and high-performance event invitation application. The core focus of this 2.0 update is a complete infrastructure refactoring to integrate a real-time database, ensuring data persistence and an immersive user experience.



### 🛠️ **What’s new in v2.0?**

* **Supabase Integration:** Migrated from a static environment to a dynamic backend ecosystem for real-time RSVP.

* **Data Security:** Implemented secure database communication. No more exposed keys; everything is now handled via **Vercel** environment variables.

* **Clean Code:** Fully refactored `script.js` to enhance readability, removing technical debt and optimizing API consumption.

* **Dynamic UI:** Improved container height logic and state animations for a smoother user experience.



### ⚙️ **Tech Stack**

* **Node.js & Express** (Backend API)

* **JavaScript (ES6+)**

* **Supabase** (BaaS)

* **HTML5 Canvas** (Starfield Effect)

* **HTML5 / CSS3** (Flexbox & Grid)

* **Vercel** (Cloud Deployment)
